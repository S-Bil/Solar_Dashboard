API-Abfrage auf Internetseite übertragen

    In diesem Beispiel wird die API-Abfrage von Postman, die Daten von der URL https://jsonplaceholder.typicode.com/users abruft, in JavaScript implementiert. Der fetch()-Aufruf wird verwendet, um die API-Abfrage auszuführen und die Daten zu erhalten. Sobald die Daten abgerufen wurden, werden sie in der Konsole ausgegeben und auf der Seite angezeigt. Das
